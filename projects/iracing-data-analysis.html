iRacing Data Analysis

Python · MoTeC i2 · Data Analysis · Telemetry · Performance Optimization | Nov 2025

Project Context

Personal development project analyzing telemetry data from 8 stints (7-8 laps each) of Ferrari 296 GT3 at Laguna Seca in iRacing. Two-part analysis: driver performance comparison with consistent setup, and setup optimization analysis with consistent driving. Data analyzed using MoTeC i2 Standard and custom Python scripts.

Project Overview

This project demonstrates two complementary approaches to performance optimization in sim racing through systematic telemetry analysis:

  • Driver Performance Analysis: Comparing driving technique across multiple stints with identical setup and conditions
  • Setup Optimization Analysis: Evaluating vehicle setup changes with consistent driver inputs

Data Collection

  • Vehicle: Ferrari 296 GT3
  • Track: Laguna Seca
  • Sessions: 8 stints totaling 7-8 laps each
  • Platform: iRacing simulation
  • Data Source: iRacing telemetry API and exported session files

Analysis Tools

  • MoTeC i2 Standard: Professional-grade telemetry analysis software for visual comparison of data channels, overlay analysis, and detailed trace inspection
  • Python Scripts: Custom data processing pipelines for automated lap analysis, statistical comparisons, and batch processing of telemetry data
  • Combined workflow leveraging MoTeC for detailed visual analysis and Python for programmatic data manipulation and metric extraction

The dual-analysis approach isolates two critical variables in racing performance: driver behavior and vehicle setup. By controlling one variable while analyzing the other, this methodology provides actionable insights for both driver development and engineering optimization.

Driver Performance Analysis

Analyzing driving technique variations across multiple stints with identical vehicle setup and track conditions. This analysis isolates driver behavior as the primary variable to identify performance patterns, consistency metrics, and improvement opportunities.

The comparison below examines performance evolution of the same driver after completing 50 laps of practice. Following systematic analysis of telemetry data and implementation of data-driven technique adjustments, measurable improvements were achieved while maintaining identical vehicle setup and conditions.

Methodology

  • Controlled variable: Fixed vehicle setup
  • Analyzed variable: Driver inputs and technique
  • Tools: MoTeC i2 for visual overlay analysis

Corner-by-Corner Analysis

Trace Legend

In all telemetry overlays below: Colored trace = Fastest lap (1:22.964) | White trace = Slower lap (1:23.617) | Delta: 0.653s improvement (0.78%)

Turn 1

Key Observations:

Braking Technique:

  • Slower lap (white) brakes significantly later and reaches 80% brake pedal position
  • Faster lap (colored) brakes earlier but at a slower pace
  • Earlier, gentler braking allows the faster lap to carry more speed into the corner with less abrupt deceleration

Mid-Corner Performance:

  • Better trail braking control in the faster lap provides increased rotation, evident in the yaw rate trace at the cursor position
  • More rotation occurs under the same steering input on corner entry, which then allows maintaining a lower average steering angle throughout the entry phase
  • Both laps achieve similar minimum speed, but the faster lap reaches it later in the corner, maintaining momentum longer

Corner Exit Understeer:

  • Both drivers experience understeer on exit, requiring them to ease off the throttle
  • Understeer confirmed by increasing steering angle while lateral G-force stalls at ~1.5G
  • Vehicle unable to generate additional lateral grip despite increased steering input

Potential Further Improvement:

  • A hypothetical case combining later, harder initial braking with better regressive braking control (progressive brake release) could yield additional performance gains
  • This would optimize both entry speed and mid-corner rotation while maintaining stability

Turn 2

Key Observations:

Braking Strategy Comparison:

  • Slower lap (white) brakes later and harder, but retains less speed on corner entry
  • Faster lap (colored) uses brake for longer duration, translating into a lower minimum speed and slightly faster exit
  • Both approaches achieve similar rotation and almost identical overall corner speed

Apex Performance:

  • The lower minimum speed achieved by the faster lap enables better rotation at the apex
  • Enhanced apex rotation directly contributes to the superior corner exit performance

Corner Exit Technique:

  • Understeer still visible, but faster lap shows more confident turn exit
  • Earlier full throttle application in the faster lap
  • Earlier return to neutral steering allows better traction management
  • Faster lap gains advantage on the following straight with better acceleration

Turn 3

Key Observations:

Braking and Turn-In Timing:

  • Both laps show very similar braking technique on corner entry
  • Slower lap (white) shows braking after turn-in
  • Faster lap (colored) demonstrates braking almost simultaneous with turn-in
  • This timing difference results in earlier and higher rotation in the faster lap

Apex Timing and Speed:

  • Faster lap achieves minimum speed much earlier, indicating an earlier apex
  • Turn 3 is followed by a relatively long straight, where the conventional approach would favor a later apex to maximize exit speed
  • Despite using an earlier apex strategy, the faster lap demonstrates that extracting the necessary rotation earlier in corner entry can overcome the traditional apex placement disadvantage
  • Enhanced early rotation allows for a confident corner exit that still capitalizes on the straight ahead

Corner Exit Performance:

  • Faster lap shows earlier throttle application as a result of the earlier braking strategy
  • Same signs of understeer present in both laps (stalling lateral G with increasing steering angle)
  • Despite understeer, the faster lap maintains advantage through better overall corner execution

Turn 4

Key Observations:

Braking Technique and Front Axle Saturation:

  • Both laps initiate braking at almost the same point with similar maximum intensity
  • Faster lap (colored) begins regressive braking earlier, resulting in much increased rotation at the cursor position
  • Higher brake pressure in the slower lap is believed to cause entry understeer
  • Critical observation: similar steering traces but vastly different lateral G and yaw rate between laps demonstrates how braking affects corner entry behavior
  • Higher brake pressure at turn-in saturates the front axle rather than relieving rear load to achieve the desired trail braking effect

Apex Performance:

  • Following the same trend as previous corners, the improved braking technique and rotation enables much faster corner entry speed
  • Both laps maintain the same minimum apex speed, showing efficiency gain comes from entry phase

Corner Exit Confidence:

  • Slower lap shows driver hesitation on throttle application timing
  • Faster lap applies throttle later but more confidently
  • Confidence stems from car already pointing in the correct direction due to superior rotation
  • Advantage maintained despite the same understeering trend present in the setup

Turn 5

Key Observations:

Braking Entry and Driver Error:

  • Faster lap (colored) shows much slower entry speed due to earlier and softer braking
  • At turn-in (marked by cursor, almost identical timing in both laps), faster lap has approximately half the brake pedal position and slower speed
  • Slower lap (white) exhibits a critical trail braking mistake
  • Steering correction required during trail braking phase in the slower lap
  • This error decreases apex speed considerably and compromises the following straight's speed

Apex and Early Throttle Application:

  • Early throttle application is crucial in this corner
  • Faster lap benefits significantly from applying throttle early

Corner Exit Imperfection:

  • Faster lap attacks the curb too harshly on exit, visible in the lateral G-force trace
  • Requires slight steering and throttle corrections on corner exit
  • Neither lap shows a perfect corner – room for improvement exists in both
  • However, the mistake on the slower lap results in much larger time loss compared to the minor exit correction needed in the faster lap

Turns 6-7 (Corkscrew)

Key Observations:

Braking Strategy and Gear Selection:

  • Slower lap (white) shows later and more aggressive braking
  • After the difference in braking points, the traces look very similar
  • The increased deceleration in the early braking phase of the slower lap appears unnecessary, as the faster driver maintains higher speed with gentler braking
  • Later braking point appears possible for the faster driver
  • Faster lap shows the last downshift happening much sooner, optimizing engine braking and gear selection
  • Slower lap's downshift occurs too late, missing optimal braking assistance

Apex Performance Through Direction Changes:

  • Minimum speed through the corners is identical in both laps
  • The steering trace when changing directions looks almost the same, showing similar technique through the elevation changes

Corner Exit and Acceleration:

  • Earlier throttle application is the key differentiator in the faster lap
  • Results in increased acceleration out of the Corkscrew
  • Despite identical minimum speeds, the timing advantage on throttle creates measurable lap time gain

Turn 8

Key Observations:

Track Characteristics and Challenge:

  • This corner is all about feel in the front axle and racing line choices
  • Critical to maintain speed through the opposite camber section, which pushes the car wide and causes significant understeer

Racing Line and Entry Strategy:

  • Faster driver (colored) closes the racing line earlier in corner entry
  • Similar turn-in point but with a more aggressive approach
  • This strategy allows staying on the gas for longer during corner entry
  • Results in higher minimum speed through the apex

Corner Exit and Setup Limitations:

  • Maximum throttle application occurs later due to corner exit understeer at this point
  • The G-force trace looks identical on corner exit between both laps, even as the faster lap's steering trace is much more aggressive – a clear sign of understeer
  • Differences between the two laps are minimal, indicating driver performance is close to the limit
  • Performance would likely benefit from a setup change toward a more oversteering car to help rotate the vehicle
  • Increasing car responsiveness is the only way to improve time in this corner – driver technique is near optimal

Turn 9

Key Observations:

Braking Technique and Banking Utilization:

  • Turn-in point (marked by cursor) is identical in both laps
  • Faster lap (colored) uses later and softer braking, allowing much more speed into the corner
  • Later downshifting also contributes to carrying more speed into the corner
  • Faster driver aims to carry more speed into the apex and make use of the banking this corner has to rotate the car
  • Slower lap shows more aggressive first phase of steering, translating into more early rotation
  • Faster driver achieves more rotation closer to the apex thanks to better braking technique

Apex Performance Trade-off:

  • Faster driver achieves a higher minimum speed thanks to added responsiveness close to the apex
  • However, struggles to get on the gas early because of that extra speed carried through the apex

Corner Exit Understeer:

  • Exit understeer visible on both laps, with steering increasing while lateral G-forces don't
  • Understeer is also influenced by the disappearing banking near the end of the corner
  • Track geometry change compounds the setup-related understeer tendency

Turn 10

Key Observations:

Braking Technique Contrast:

  • Slower driver (white) brakes later and much more aggressively, decreasing speed too much
  • The braking trace on the regressive phase is much less smooth in the slower lap
  • Steering is more aggressive in the slower lap

Corner Shape Strategy and Apex Speed:

  • Slower lap shows a more V-shaped corner approach
  • Faster lap (colored) focuses on maintaining apex speed with a U-shaped approach
  • Faster lap achieves better entry rotation thanks to better trail braking technique
  • Faster lap achieves better speed through the corner itself

Exit Strategy Trade-off:

  • Corner exit is better for the slower lap due to the smaller minimum speed (marked by cursor)
  • Lower minimum speed translates into earlier throttle application and faster exit
  • This is particularly beneficial in this corner as it precedes the longest straight of the track
  • The faster lap's U-shaped corner aim loses its advantage over the next straight
  • Critical lesson: V-shaped approach with lower minimum speed optimizes exit velocity for long straights

Setup Optimization Analysis

Evaluating the impact of vehicle setup changes with consistent driver performance. This analysis isolates setup variables to quantify their effect on lap times, handling balance, and overall vehicle behavior.

Methodology

  • Controlled variable: Consistent driving inputs and technique
  • Analyzed variable: Vehicle setup parameters (suspension, aerodynamics, differential)
  • Key metrics: Lap time deltas, corner speed changes, stability indicators, tire performance
  • MoTeC i2: Direct comparison of identical corner entries with different setups, suspension travel and damping analysis, tire temperature distribution patterns
  • Python: Quantitative setup change correlation with lap time improvements, automated detection of setup-sensitive corners, statistical validation of setup effects
